### Run Locally

```
vagrant up
vagrant ssh
cd /vagrant
honcho start
```

Then, you are expected to see the following json response after browsing http://0.0.0.0:5000

```
{"name":"Customer Service API","paths":"http://0.0.0.0:5000/","version":"1.0"}
```
